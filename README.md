# rPhobos
A very simeple rust based ring 3 Windows rootkit for hiding files, processes and registry keys. <br>
This project is heavily inspired by [r77](https://github.com/bytecode77/r77-rootkit/). <br>

> [!WARNING]  
> This project serves only to help me better understand and learn about hooking functions in Windows, i do not intend for this to be as good or hardened as r77.
